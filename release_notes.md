# 🚀 OAAI Model-Radar v1.0.0 稳定版发布！

## 📝 项目简介
**OAAI Model-Radar** 是一款通用的 LLM API 节点管理器与测速工具，完美适配各类 Agent 框架。它可以帮助开发者轻松管理多个大模型 API 密钥，实时监测各个节点的连通性与响应速度，全面提升 AI 开发和调试的效率。

## 🌟 核心特性
*   🐍 **纯 Python 零依赖**：无须繁琐地安装额外的第三方库，只需 Python 基本环境，真正做到开箱即用。
*   ⚡ **并发极速测速**：支持一键并行测试所有大模型节点的可用性与响应时长，秒级获取结果，高效便捷。
*   📂 **多配置文件管理**：支持多套 API 节点配置方案的独立管理和一键切换，轻松应对不同项目及环境。
*   🌐 **流畅的双语支持**：内置完善的中英双语切换，满足全球各地开发者的使用需求。
*   🎨 **极致视觉美学**：采用现代化的高颜值玻璃拟物化设计（Glassmorphism），并提供深邃优雅的暗色模式体验。
*   🐳 **全平台轻便部署**：支持本地脚本一键启动，同时也提供完整的 Docker 容器化支持，部署轻而易举。

## 🛠️ 简单快速的安装与运行

### ✨ 选项 A：本地一键启动（推荐）
```bash
# 1. 克隆项目到本地
git clone https://github.com/gangke3/OAAI-Model-Radar.git
cd OAAI-Model-Radar

# 2. 启动服务
# Windows 用户请直接双击运行根目录下的 start.bat
# Mac/Linux 用户请运行：
bash start.sh
```
> 服务启动后，浏览器将会自动打开访问本地地址即可使用。

### 🐳 选项 B：Docker 容器部署
```bash
# 一键使用 Docker Compose 启动服务
docker-compose up -d
```
